## 0. 原則（=この文書を貫くルール）

- **宣伝＝実装の一致**: 宣伝してよいのは**実装済み機能のみ**。詳細分析（曜日/時間帯/SNS別/月次/予測）は **v1.1送り**のため、ストア文言・スクショに**一切登場させない**（=景表法・Apple 3.1.2 誇大表示リスク回避。CEO裁定 2026-06-19）。
- **「広告削除」は副次特典**: AdMob バナー広告を実装済（無料プランのみ表示・プレミアムは非表示）。「広告なし」を前面の主訴求にはせず（主訴求は収集体験・限定Mofi・プレミアム卵）、プレミアム特典の一項目として「広告を非表示に／快適にプレイ」程度に記載（=実装と一致 / PRICING §2）。データ安全性は広告ID収集を申告済（STORE_DATA_SAFETY §2）。
- **景表法・審査の禁止表現**: 根拠のない数字（「○万人が利用」「満足度98%」）、断定・誇大（「絶対」「100%」「No.1」「世界初」）、他社アプリ名の比較記載は**書かない**。
- **スクショ内の価格/「無料」/「割引」/トライアル表記は禁止**（=Apple 審査 2.3.7。ストアは「無料」も価格参照と判定しリジェクト）。価格は**説明文にのみ**記載（=説明文は公式にOK）。
- **対象SNS4アプリ**（TikTok / Instagram / YouTube / X）は PRD S3 で確定。文言でも「主要SNS」と表現し、ロゴの無断使用はしない（=スクショは自前アイコン/抽象表現で）。

## 2. キーワード（App Store 100字 / Google Play は説明文織り込み）

### 2-1. App Store キーワードフィールド（100字・カンマ区切り）

> 鉄則: アプリ名・サブタイトルと**重複させない**（重複は枠の無駄）。スペースは入れず単語区切りのみ。検索意図＝「スクリーンタイム / 習慣化 / 集中 / 収集・育成 / SNS依存」をカバー。

```
スクリーンタイム,スマホ依存,デジタルデトックス,習慣化,集中,勉強,作業,ポモドーロ,自己管理,目標,育成ゲーム,収集,コレクション,キャラ,ペット,たまご,図鑑,かわいい,癒し,継続,記録,時間管理,やめたい,だらだら,SNS断ち
```

- 字数: 約97字（=100字枠内）。
- 選定理由:
  - **検索ボリューム狙い**: スクリーンタイム / スマホ依存 / デジタルデトックス / 習慣化 / 集中 / 勉強 / 時間管理（=Forest・Opal系の主要流入語）。
  - **収集ゲーム意図**: 育成ゲーム / 収集 / コレクション / キャラ / ペット / たまご / 図鑑（=差別化の核を検索面でも拾う）。
  - **感情・課題語**: かわいい / 癒し / やめたい / だらだら / SNS断ち（=ユーザーの自己検索ワード）。
- アプリ名で既出の「Moffy / SNS / 卵 / 育てる」、サブタイトルの「スマホ時間 / 収集」は**ここに入れない**（重複回避）。

### 2-2. Google Play キーワード方針（=専用フィールドなし。説明文に自然に織り込む）

- Playは独立キーワード欄がなく、**タイトル+短い説明+説明文の本文**から検索インデックスされる。よって**キーワードの詰め込み（=不自然な羅列）はスパム判定リスク**。自然な文章に主要語を**2〜3回ずつ**織り込む。
- 優先織り込み語（説明文に自然分散させる）:
  - 主軸: `スクリーンタイム` / `スマホ時間` / `SNS` / `習慣化` / `デジタルデトックス`
  - 収集: `育成` / `収集` / `図鑑` / `コレクション` / `かわいいキャラ` / `卵` / `孵化`
  - 行動: `集中` / `勉強` / `時間管理` / `減らす` / `続けられる`
- NG: 同じ語の機械的連呼、競合アプリ名（Forest / Finch / Opal 等）の記載（=規約・景表法リスク）。

## 6. ローカライズ / ASO 運用メモ

### 6-1. A/Bテスト候補（=launch後に回す。優先順）

| 優先 | テスト対象 | バリアント | 計測指標 | 備考 |
|---|---|---|---|---|
| 1 | **スクショ1枚目** | A価値訴求 / B因果訴求 / C収集訴求（§4） | ストアページ閲覧→DLのCVR（=コンバージョン率） | 最もインパクト大。1枚目はファネル先頭 |
| 2 | **アプリ名の説明部** | 「SNS減らして卵を育てる」/「集めて減らすSNS時間」 | 検索流入→DL率・検索順位 | Playの「ストアの掲載情報のテスト」機能を使用 |
| 3 | **短い説明（Play）** | 価値文 / 収集語厚め文（§1-2） | 一覧→詳細→DL率 | Playのみ。Apple はサブタイトルで代替 |

# Moffy 見た目の方向（確定：C土台＋Aの目印 / 2026-06-30）

デザイン部署＋マーケ部署の合議で決定。3案（A絵本 / B クリーン / C ジオラマ）を比較し、
**「C（上質ソフト3Dジオラマ）を土台に、A（コージー絵本）の手描きインク輪郭を“Moffiだけの目印”
として1本足す」ハイブリッド**を採用。**Bは両部署とも「量産の癒し系に埋もれる」で不採用**。

## なぜこの合わせ技か
- **C土台**＝今のつや卵アセット・トークンをそのまま活かせ、卵12種×段階＋Mofi30種を**同一光源で
  完成させられる**唯一の現実解（ソロ開発の完遂性）。作業量最小で上質感。
- **Aの目印（インク輪郭）**＝Cを素で出すと「AI量産3D顔の亜種」に寄るため、**卵/Mofiのシルエットに
  手描き風の焦げ茶インク輪郭を1本**足して、一目でMoffiと分かる差別化を担保（デザイン懸念＋マーケの
  サムネ差別化を同時に解決）。
- **store用（アイコン/スクショ1枚目）はA寄りの温かさ**＝ストア一覧で親指を止める（マーケ）。

## 確定ルール（作り始めの1枚）

### パレット（＝現行トークンをC基準に。ほぼ現状踏襲）
- bg=クリーム `#FBF4E6` / surface=白寄りクリーム `#FFFDF8` / ink=焦げ茶 `#3A322B`
- primary=オレンジ `#FF8A3D` / accent=深緑 `#3E9A5E` / nest(台)=砂金 `#E7C892`
- レア色は `RarityToken`（common霧緑 / rare水色 / **sr紫はここだけ許可** / ssr金）。glow=内側からの淡い発光。
- **量産SNSの青紫グラデは背景主役にしない**（空色は上端のごく一部のみ）。

### シグネチャ（Moffiだけの目印）
- **卵・Mofi のシルエットに焦げ茶インク `#4A3B2E` の手描き風輪郭を1本**（外周に細く）。
  実装: 卵アセットに焼き込み済み（`assets/images/egg/*.png`）。将来アセット差し替え時も同じ処理を通す。
- 署名要素「巣リング」（砂色の円台＋地面楕円影）は継続。**「砂リング（額縁）＋藁の巣（卵のおうち）」の
  二層ジオラマ**に卵を1つ置く構図を全画面で反復（隠してもMoffと分かる核）。

### 光・影・質感（ジオラマの一貫性）
- **光源は左上・単一**で全アセット統一（つや玉の上1/3ハイライト、下1/3に台色の反射光）。
- 影は真っ黒禁止＝**巣色の焦げ茶（bark系）**を薄く2段（近影ぼかし8 / 浮遊16）。要素は必ず地面影で接地。
- 角丸2段（12 / 24）＋pill。境界線は最小、**浮かせは影で**（Bのフラットにも A の輪郭にも寄せすぎない）。

### 成長・演出
- 卵は成長で**横向きのヒビ**を段階的に重ねる（`_CrackOverlayPainter`）。将来は段階別イラストに差し替え可
  （`docs/EGG_ART_PROMPT.md`／段階画像は輪郭処理を通す）。
- 動きは「ゆっくり・やわらか・重さ」。待機は呼吸するような上下、孵化・色違いだけ光を強めて格上げ。

## 進め方（マーケの二段構え）
1. **本体はCで即・完成**（今の卵＋インク輪郭で全画面を揃える）→ 初日から計測・レビュー獲得。
2. **アイコン/スクショ1枚目はA寄りの温かさ**で先行制作（ストア突破率）。→ 実装済みアイコンは
   輪郭入り卵＋温かいクリーム→テラコッタ背景に更新。
3. 余力が出たら段階的に A（マット水彩）へ寄せる「憧れ枠」を残す。

## 未対応・次
- Mofi は現状 Material アイコン（種族別）。個体イラスト化の際に同じインク輪郭・同一光源を適用。
- 影の2段化・光源統一を各 widget に反映（`nest_panel` / カード）。
- ストアのスクショ一式（1枚目=A世界観）。
